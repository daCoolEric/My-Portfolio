My present experience enables me to build webpages, web apps and native apps from scratch using HTML, CSS, JavaScript, React.js, Redux, Node.js, Express.js and many more. I'm a very vivid visualizer who believe that if you can see it, you can create it. I spend most of my time thinking and visualizing the solution to a problem before I start coding or implementing it.
